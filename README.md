# Downloader MP3 Converter

A simple and efficient tool for downloading and converting videos to MP3 format using [ffmpeg](https://ffmpeg.org/) and [yt-dlp](https://github.com/yt-dlp/yt-dlp).

## Requirements

Before using the script, make sure to install the following dependencies:

1. **Install ffmpeg**  
   Recommended version: `ffmpeg-8.1-essentials_build`. Download and install ffmpeg from the [official website](https://ffmpeg.org/download.html).

2. **Install Python**  
   Python 3.x is required to run this script.

____

## Installation and Setup

### 1. Clone the repository

```bash
git clone https://github.com/username/repository-name.git
cd repository-name
```

### 2. Create a virtual environment

**Linux/MacOS**
```bash
python -m venv venv
venv\Scripts\activate
```

**Windows**
```bash
python -m venv venv
venv\Scripts\activate
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
pip install yt_dlp
```

### 4. Run the script

```python
python converter_mp3.py
```
